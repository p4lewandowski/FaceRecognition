FaceRecognition_plotting.show_found_face(image_s, data, cnn=False, id=-1)[źródło]¶

Funkcja wyświetla w oddzielnym od aplikacji GUI oknie twarz przekazaną w argumentach w celu porównania szukanej i znalezionej twarzy.

Args:

image_s: Szukana twarz w skali szarości bądź RGB.

data: Zależna od wartości cnn.

Gdy cnn = True - przekazywane są dwa parametry: 1. Szukane zdjęcie. 2. Lista o trzech elementach zawierająca prawdopodobieństwa przynależności do danej klasy, baza danych z nieprzetworzonymi zdjęciami twarzy, etykiety do bazy danych zdjęć, umożliwiające połączenie prawdopodobieństw przynależności do zdjęć w bazie. Gdy cnn = False - Jedynie szukane zdjęcie.

cnn: Argument opcjonalny determinujący czy zdjęcia ma być szukane z użyciem danych z sieci neuronowej, czy wyświetlone ma być znalezione zdjęcie dla twarzy własnych.
